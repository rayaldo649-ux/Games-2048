Cara Bermain Game 2048
Geser papan menggunakan tombol panah (↑ ↓ ← →) untuk menggerakkan semua ubin ke arah tersebut.
Jika dua ubin dengan angka yang sama bertemu, keduanya akan bergabung menjadi satu angka yang lebih besar.
Setiap kali kamu menggeser, akan muncul ubin baru (2 atau 4) di posisi acak.
Lanjutkan menggabungkan angka untuk mencapai nilai 2048.
Jika papan penuh dan tidak ada ubin yang bisa digabungkan lagi, permainan dianggap kalah.
